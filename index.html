Getting started with Gamemacs

Get directly to the Gamemacs superproject on Github and check it out! Instructions on this site.

The Gamemacs project is meaning to build upon existing projects (written in Common Lisp) aiming to create a game creation IDE which provides a great graphical environment to model your resources as well as other tools to get creating games for whatever system as easy as possible.
